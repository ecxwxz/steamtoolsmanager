# steamtoolsmanager
Steam Manifest 下载器 &amp; 自动入库工具的集成，旨在帮助没有任何基础的小白学习steam的运行机制，解放双手自动入库
## 功能
'''
-1.遍历manifest仓库,下载lua以及其他文件
-2.下载后的文件处理，自动按照游戏名称归类，方便阅读
-3.自动入库(目前支持steamtools的入库方式)

'''
## 依赖
本程序依赖于steamtools的解锁方式，仓库中也存放了安装包 [steamtools官网](https://www.steamtools.net/)
APP id 的获取方式 steam官方商店里的数字就是appid，也访问 [steamui](https://steamui.com/)
