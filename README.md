# steamtoolsmanager

> 🛠 **Steam Manifest 下载器 & 自动入库工具整合版**  
> 旨在帮助零基础用户了解 Steam 的基本运行机制，并减少重复手动操作。

---

## ✨ 功能概览

1. **遍历 Manifest 仓库**  
   - 批量下载对应的 `.lua` 以及其他相关文件。

2. **下载文件自动整理**  
   - 按照 **游戏名称** 自动归类至对应文件夹，方便浏览与管理。

3. **自动入库支持**  
   - 目前支持基于 **steamtools** 的入库方式（需自行安装与配置）。

---

## 📦 依赖说明

本程序依赖于 steamtools 的相关机制进行解锁与入库，使用前请自行了解相关原理与风险。

- steamtools 安装包已包含于本仓库中：  
  👉 [steamtools 官网](https://www.steamtools.net/)

- APP ID 的获取方式：  
  - 在 Steam 官方商店页面中，URL 里的数字部分即为该游戏的 `APPID`  
  - 或使用第三方站点：👉 [steamui](https://steamui.com/)

> ⚠ 请务必遵守 Steam 用户协议及相关法律法规，合理、合规地使用本工具与依赖软件。
